## 数据理解

### 数据质量探查

- 一般在SQL里面查某些字段用和总体的缺失情况

  ~~~
  select count(1) from table where name is null
  ~~~

  

### 重要特征探查

- 选择思路：流失客户的数据一般是不稳定的，流失客户的后期数据与前期数据会有差异等，在业务理解里面已有分析
